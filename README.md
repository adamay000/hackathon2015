hackathon2015
==

___

####Getting Started
```
git clone https://github.com/adamay000/hackathon2015.git
npm install
bower install
```

####Run

For test: `gulp dev`

For release: `gulp release`

####Tools

- gulp
- browserify
- ES6(babelify)
- sass(gulp-sass)

####Directory Structure

```
├ app
├ gulp
│ └ tasks
├ src
│ ├ css
│ ├ js
├ release
├ tmp
```
